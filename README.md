# Council Chambers

A web-based game where you take on the role of a city council member and make decisions that affect the city's future.

## Description

Council Chambers is a single-player, story-driven game where you are a newly elected city council member. You will be presented with a series of scenarios and must make decisions that impact the city's budget, public trust, and more. The game is built with React, TypeScript, and Framer Motion.

## Features

*   **Engaging Storyline:** A captivating story that unfolds based on your decisions.
*   **Meaningful Choices:** Make decisions that have a real impact on the game's outcome.
*   **Modern Design:** A sleek and modern user interface with a digital courtroom vibe.
*   **Smooth Animations:** Smooth and performant animations powered by Framer Motion.
*   **Admin Panel:** A built-in admin panel that allows you to customize the game's scenarios.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   [Node.js](https://nodejs.org/en/)
*   [npm](https://www.npmjs.com/)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/neiloy-neil/Councils-Chambers.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Usage

1.  Start the development server
    ```sh
    npm run dev
    ```
2.  Open your browser and navigate to `http://localhost:3000`
